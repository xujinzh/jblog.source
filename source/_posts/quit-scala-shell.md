当运行 flink scala shell 时，想要退出本地命令行端口，可以使用如下的命令：

<!--more-->

```scala
:q
:quit
sys.exit
```

也可以使用快捷键：
- 在 Linux 系统下
Ctrl + d
- 在 Windows 系统下
Ctrl + z
